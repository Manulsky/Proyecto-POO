Proyecto POO

Este codigo utiliza una serie de interfaces para crear un videojuego
donde el objetico principal es de esquivar o destruir los enemigos que 
aparecen en pantalla y se desplazan hasta el jugador.

El impactar un enemigo con el jugador da un GameOver

El despejar la zona da un You Win

El codigo contiene 3 niveles, varian en la cantidad de enemigos,
velocidad del desplazamiento de los enemigos asi como del jugador.
